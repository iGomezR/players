Basic implementation to react + redux and jest.


## Getting Started

Clone, install and run.

    # clone it
    git clone https://github.com/iGomezR/players.git
    cd players
    
    # Install dependencies
    npm install || yarn

    # Start production server:
    npm start

    # Run test:
    npm test
    
    # And go to
    http://localhost:3000
